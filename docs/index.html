The GeoPackage Android Map SDK was developed as part of the GeoPackage Libraries by the National Geospatial-Intelligence Agency (NGA) in collaboration with BIT Systems.

GeoPackage Android Map is a SDK implementation of the Open Geospatial Consortium GeoPackage spec. It is listed as an OGC GeoPackage Implementation by the National Geospatial-Intelligence Agency.

The GeoPackage Map SDK provides Google Map library implementations in addition to the inherited base GeoPackage Android functionality. The library provides TileProvider overlay implementations and Google Map shape translations.
